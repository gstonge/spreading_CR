spreading_CR.SpreadingProcess.__init__¶

SpreadingProcess.__init__(self: spreading_CR.SpreadingProcess, edge_list: List[Tuple[int, int]], transmission_rate: float, recovery_rate: float, waning_immunity_rate: float) → None¶

This is the constructor of the class.

Parameters:
  • edge_list – Edge list of the network.
  • transmission_rate – Rate of transmission per edge.
  • recovery_rate – Rate of recovery of infected nodes.
  • waning_immunity_rate – Rate for immunity loss. For the SIR model the rate must be set to 0. For the SIS model, set it to numpy.inf.
